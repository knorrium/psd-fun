psd-fun
-------

fun with psd files

requirements
------------

- psd.rb

- awesome_print

installing
----------

if you have a Ruby manager like rbenv or rvm

```
$ bundle install
```

or to install the required libraries system-wide

```
$ sudo gem install awesome_print psd
```

running
-------

```
$ ruby fun.rb
```

each layer will be saved to the `output` dir
